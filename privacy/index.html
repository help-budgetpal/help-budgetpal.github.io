Privacy Policy

BudgetPal is a desktop application that helps users import bank transaction data from CSV files and automatically populate and categorize this data in a Google Sheets document they choose.

What Data We Access

  • Google Drive File Access: We request access to create, open, and modify Google Sheets files that you explicitly select using the Google File Picker. This access is limited to only those files you choose.
  • Basic Profile Info: We request your email address and basic profile information solely for user identification and personalization within the app.

What We Don't Do

  • We do not collect or store your data on any external servers.
  • We do not access or modify any files on your Google Drive unless you explicitly select them via the Google File Picker.
  • We do not share your personal or financial data with any third parties.
  • All data processing happens locally on your device.

Security

BudgetPal uses Google's secure OAuth 2.0 system for authorization. You are always in control of which files and data you authorize.

Contact

If you have any questions or concerns, please reach out to help.budgetpal@gmail.com.
